## Tips and guidelines:
Filesystem structure of a Python project: https://as.ynchrono.us/2007/12/filesystem-structure-of-python-project_21.html

```
Project/
|-- bin/
|   |-- project
|
|-- project/
|   |-- test/
|   |   |-- __init__.py
|   |   |-- test_main.py
|   |   
|   |-- __init__.py
|   |-- main.py
|
|-- setup.py
|-- README
```
